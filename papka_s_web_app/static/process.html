Марковские цепи

|Общая информация|История|Цепи

Цепь Маркова — это последовательность случайных событий с конечным числом исходов,

где вероятность наступления каждого события зависит только от состояния, достигнутого в

предыдущем событии. Цепь Маркова — это последовательность состояний, каждое из кото-

рых берётся из пространства дискретных состояний, удовлетворяющее марковскому свойству

и переходы из одного состояния в другое происходят через равные либо через произвольные

промежутки времени.

  • Если множество значений дискретное, то – цепь маркова.
  • Если время дискретное – цепь маркова с дискретным временем
  • Если время непрерывное – цепь маркова с непрерывным временем.
